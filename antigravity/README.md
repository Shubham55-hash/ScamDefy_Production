# ScamDefy Antigravity — Testing Framework

## Quick Start

```powershell
# From project root (ScamDefy-Final/)

# 1. Install test dependencies
cd api
pip install pytest pytest-asyncio httpx pytest-json-report

# 2. Train the local voice model (one-time, ~60s)
python scripts/train_voice_model.py

# 3. Run the full test suite
cd ..
python antigravity/runner.py
```

## Interactive Test Lab (Frontend)

Activate Test Mode in your browser:

```js
// In DevTools console:
localStorage.setItem('sd_testmode', '1');
location.reload();
```

Or append `?testmode=1` to the URL. A **⚗ Test Lab** nav item appears.

Click **Run All Tests** to run all 21 live tests against the deployed API.

## Voice Detection

The voice detection pipeline uses **3 tiers**:

1. **Local Model** (always available, no internet) — 24-feature acoustic classifier
   - Pitch jitter, shimmer, spectral flatness variance, MFCC delta smoothness, pause regularity
   - GradientBoosting trained on 3000 synthetic samples

2. **HuggingFace** (`MelodyMachine/Deepfake-audio-detection-V2`) — downloaded on first use
   - wav2vec2 fine-tuned on deepfake detection datasets
   - Weight: 45% (wideband), 25% (telephony/narrowband)

3. **Gemini 2.0 Flash** — multimodal LLM analysis
   - Analyses naturalness, breathing, micro-pauses
   - Weight: 25% (wideband), 40% (telephony)
